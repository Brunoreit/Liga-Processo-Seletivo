# MVP Scope

## Overview

The MVP focuses on creating a centralized system to help users manage their job applications and interview processes, replacing scattered notes and spreadsheets with a structured career tracking workflow.

## MVP Goals

The main goals of the MVP are:

- Provide a centralized place to track job opportunities and application progress.
- Allow users to manage their interview process history.
- Create a foundation for future career management features.
- Validate the usefulness of treating career development as an organized process.

## Core Features
### 1. User Management
Features:
   - User registration
   - User authentication
   - User profile management

### 2. Job Application Management
Features:

- Create a job application 
- View all applications
- Update application information
- Delete an application
- Change application status

#### Job data:
1. tittle
2. Company
3. Position
4. job URL
5. Description
6. Application date
7. Status
8. Notes

#### Initial status:
- APPLIED
- INTERVIEW
- OFFER
- REJECTED
- GHOSTING

### 3. Interview Management
Features:
- Store interview dates
- add notes and observations

### 4. Basic Dashboard
Features:
- Total applications 
- Applications by status 
- Active processes 
- Interview count
